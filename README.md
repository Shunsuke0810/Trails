# README

## 開発言語
---
- Ruby 3.0.1
- Ruby on Rails 6.0.5

## 就職Termの技術
---
- お気に入り機能
- Ajaxを使ったコメント機能

## カリキュラム外の技術
---
- gem ransacを使用したラベル検索機能

## 実行手順 
---

```
$ git clone git@github.com:Shunsuke0810/Trails.git
$ cd Trails
$ bundle
$ rails db:create && rails db:migrate
$ rails s
```
